<h3 align="center">A new syntax for C &amp; C++, in one header file</h3>
</p>

<br />

sea is a new syntax for C &amp; C++. It can be used by adding the following line of code to a `.c`, or `.cc`/`.cpp` file:

```cpp
#include <sea.h>
```
